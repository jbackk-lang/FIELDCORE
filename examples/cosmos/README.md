# Przykład: Model kosmosu w FIELDCORE

Ten przykład pokazuje:

- jak uruchomić toruso_mobius.py,
- jak interpretować skręt Λ–τ–ρ–J,
- jak klasyfikować galaktyki według fazy skrętu.

## Uruchomienie

python3 toruso_mobius.py

## Wynik

Powstaje trójwymiarowy model toruso‑Möbiusa, który reprezentuje globalną strukturę pola.
